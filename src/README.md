# Исходные коды проекта

Структура директорий:

- `auxiliary` - вспомогательные скрипты.
- `benchmark` - основной код тестовой системы.
- `converter` - конвертер моделей (использует компонент Model Optimizer).
- `inference` - скрипты для тестирования глубоких моделей в синхронном
  и асинхронном режимах.
